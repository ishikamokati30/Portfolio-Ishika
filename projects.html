CityServe

CityServe is a city-centric local marketplace web platform designed to help users discover nearby shops, browse products and services, and place orders with ease. Users can search and filter shops by category, view shop details such as timings, contact and address, add items to a cart, apply coupons and discounts, choose delivery options, make payments, and track order status. The system also supports user authentication, order history, dynamic pricing with discounts, and real-time shop availability, making local shopping simple and digital.

Python • Django • HTML • CSS • JavaScript • SQLite
